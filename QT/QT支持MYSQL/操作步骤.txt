1.确保已安装MySQL数据库，并记下MySQL安装路径。
https://downloads.mysql.com/archives/installer/

2.打开Qt的命令行工具（例如Qt Creator的“终端”窗口）。

3.导航到Qt安装目录的源代码目录，并进入sqldrivers目录。例如，如果Qt安装在/home/user/Qt目录下，则可以执行以下命令：
cd /home/user/Qt/6.3.0/Src/qtbase/src/plugins/sqldrivers/

4.在sqldrivers目录中，找到mysql目录。如果没有该目录，则需要手动添加。

5.在mysql目录中创建一个  mysql.pro 文件，并将以下内容复制到该文件中：

TEMPLATE = lib
TARGET = qsqlmysql
CONFIG += plugin
QT += sql-private
INCLUDEPATH += "MySQL安装路径/include"
LIBS += -L"MySQL安装路径/lib" -lmysqlclient
SOURCES += main.cpp qsql_mysql.cpp qsql_mysql_p.h
HEADERS += qsql_mysql_p.h
将"MySQL安装路径"替换为实际的MySQL安装路径。

保存并关闭.mysql.pro文件。

6.在命令行中执行以下命令，编译MySQL驱动程序：
qmake "mysql.pro"
make

7.编译成功后，将生成的库文件（例如libqsqlmysql.so或qsqlmysql.dll）复制到你的Qt项目中的合适位置。

8.在Qt项目的.pro文件中，添加以下行以链接MySQL驱动程序的库文件：
LIBS += -L"驱动程序库文件所在路径" -lqsqlmysql
将"驱动程序库文件所在路径"替换为实际的MySQL驱动程序库文件所在的路径。

9.重新编译和运行项目，确保MySQL驱动程序被正确加载
