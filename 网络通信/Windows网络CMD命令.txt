ipconfig 查看本机IP及mac地址等
ipconfig /release 断网重连，并显示最新IP地址等信息


/////////////////////////////////////////////////////////
ping	检测网络是否畅通
用法：
1.ping + IP 			（ping 192.168.1.1）
2.ping + 主机域名		（ping www.baidu.com）

-t 不断发送数据包
-l 定义数据包大小
-n 定义发送数据包次数

netstat 查看本机网络连接和监听端口 (netstat -ano)

-a 	查看所有
-p	查看指定协议类型的连接，如 -p TCP，意为只看TCP连接
-b  查看时带上 相关的exe或组件分类
-o	查看时带上 相关的进程ID
-r	查看本机网管、子网掩码等

tracert  跟踪本机到目标计算机传输信息时所有路由信息
用法：
tracert + IP 		 （tracert 172.16.149.218）



//////////////////////////////////////////////////////////
net
用法：
1.net view + IP 					（查看目标计算机有哪些共享资源）
  net share							（查看本机的共享资源）
2.net use + 盘符名:IP + 共享名		（映射网络共享资源为本地盘）
3.net start + 服务名称				（远程开启目标计算机的服务，前提是已建立连接）
4.net stop + 服务名称				（远程关闭目标计算机的服务）
5.net user 							（查看当前计算机的用户账户，可加参数进行管理用户）
6.net localgroup					（查看当前计算机的用户组，可加参数进行管理用户）

//////////////////////////////////////////////////////////
telnet
用法：
telnet + IP + 空格 + PORT  					 （检测目标计算机指定端口是否可连接）

相关：
mstsc			桌面远程连接(远程桌面)
telnet			终端远程连接
